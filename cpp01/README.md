# CPP Module 01

Subject focus: memory allocation, pointers to members, references and switch statements.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | BraiiiiiiinnnzzzZ | `BraiiiiiiinnnzzzZ` | Implement `Zombie`, `newZombie` and `randomChump` to compare heap and stack allocation. |
| ex01 | Moar brainz! | `Moar_brainz!` | Allocate and initialize a horde of zombies. |
| ex02 | HI THIS IS BRAIN | `HI_THIS_IS_BRAIN` | Print a string address/value through a variable, pointer and reference. |
| ex03 | Unnecessary violence | `Unnecessary_violence` | Model `Weapon`, `HumanA` and `HumanB` with references and pointers. |
| ex04 | Sed is for losers | `Sed_is_for_losers` | Replace occurrences of one string with another in a file, without `std::string::replace`. |
| ex05 | Harl 2.0 | `Harl_2.0` | Dispatch complaint levels through pointers to member functions. |
| ex06 | Harl filter | `harlFilter` | Filter complaint output by level using `switch`. |

## Build

```sh
cd ex04
make
./Sed_is_for_losers file old new
make fclean
```
