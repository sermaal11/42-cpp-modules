# CPP Module 03

Subject focus: inheritance.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Aaaaand... OPEN! | `Aaaaaa..._OPEN` | Implement the base `ClapTrap` class. |
| ex01 | Serena, my love! | `Serena,_my_love` | Add `ScavTrap`, derived from `ClapTrap`. |
| ex02 | Repetitive work | `Repetitive_work` | Add `FragTrap`, another derived trap type. |
| ex03 | Now it's weird! | `Now_it_is_weird` | Implement `DiamondTrap` with multiple inheritance from `ScavTrap` and `FragTrap`. |

## Build

```sh
cd ex03
make
./Now_it_is_weird
make fclean
```
