# CPP Module 05

Subject focus: repetition and exceptions.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Mommy, when I grow up, I want to be a bureaucrat! | `mommy_when_i_grow_up_i_want_to_be_a_bureaucrat` | Implement `Bureaucrat` with grade bounds and exceptions. |
| ex01 | Form up, maggots! | `form_up_maggots` | Add signable `Form` objects. |
| ex02 | No, you need form 28B, not 28C... | `no_you_need_form_28b_not_28c` | Add `AForm` and concrete executable forms. |
| ex03 | At least this beats coffee-making | `at_least_this_beats_coffee_making` | Implement `Intern` as a form factory. |

## Build

```sh
cd ex03
make
./at_least_this_beats_coffee_making
make fclean
```
