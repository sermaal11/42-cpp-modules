# CPP Module 02

Subject focus: ad-hoc polymorphism, operator overloads and Orthodox Canonical Form.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | My First Class in Orthodox Canonical Form | `My_first_class_in_orthodox_canonical_form` | Implement a first `Fixed` class in Orthodox Canonical Form. |
| ex01 | Towards a more useful fixed-point number class | `Towards_as_a_more_used_fixed-point_number_class` | Add integer/float constructors and conversion functions. |
| ex02 | Now we're talking | `Now_we_are_talking` | Add comparison, arithmetic, increment/decrement and min/max operators. |
| ex03 | BSP | `BSP` | Implement a point-in-triangle test with `Point`, `Fixed` and `bsp`. |

## Build

```sh
cd ex03
make
./BSP
make fclean
```
