# CPP Module 04

Subject focus: subtype polymorphism, abstract classes and interfaces.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Polymorphism | `Polymorphism` | Implement `Animal`, `Dog`, `Cat`, `WrongAnimal` and `WrongCat` with virtual behavior. |
| ex01 | I don't want to set the world on fire | `I_dont_want_to_set_the_world_on_fire` | Add `Brain` ownership and verify deep copies. |
| ex02 | Abstract class | `Abstract_class` | Make the animal base abstraction non-instantiable. |

## Subject note

The PDF also defines `ex03: Interface & recap`; this repository currently contains `ex00` to `ex02`.

## Build

```sh
cd ex02
make
./Abstract_class
```
