# CPP Module 00

Subject focus: namespaces, classes, member functions, stdio streams, initialization lists, static and const basics.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Megaphone | `megaphone` | Print command-line arguments in uppercase, or the default feedback noise when no argument is passed. |
| ex01 | My Awesome PhoneBook | `MyAwesome` | Implement an 80s-style phonebook with `ADD`, `SEARCH` and `EXIT`. |
| ex02 | The Job Of Your Dreams | `Account` | Recreate the expected behavior of the provided `Account` test setup. |

## Build

```sh
cd ex00
make
./megaphone "hello world"
make fclean
```
