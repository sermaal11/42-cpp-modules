# CPP Module 08

Subject focus: templated containers, iterators and algorithms.

## Module-specific rule

The subject explicitly expects the STL containers and algorithms to be used in this module.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Easy find | `Easy_find` | Implement `easyfind` for sequential STL containers. |
| ex01 | Span | `Span` | Store up to `N` integers and compute shortest/longest spans. |
| ex02 | Mutated abomination | `mutated_abomination` | Implement an iterable `MutantStack`. |

## Build

```sh
cd ex01
make
./Span
make fclean
```
