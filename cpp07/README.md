# CPP Module 07

Subject focus: C++ templates.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Start with a few functions | `start_with_a_few_functions` | Implement function templates `swap`, `min` and `max`. |
| ex01 | Iter | `Iter` | Implement `iter`, a function template that applies a function to each array element. |
| ex02 | Array | `Array` | Implement a class template `Array` with bounds-checked access. |

## Build

```sh
cd ex02
make
./Array
make fclean
```
