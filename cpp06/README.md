# CPP Module 06

Subject focus: C++ casts.

## Exercises

| Exercise | Official title | Program | Subject requirement |
| --- | --- | --- | --- |
| ex00 | Conversion of scalar types | `convert` | Convert a scalar literal to `char`, `int`, `float` and `double`. |
| ex01 | Serialization | `Serialization` | Serialize and deserialize a `Data*` through `uintptr_t`. |
| ex02 | Identify real type | `identify_real_type` | Generate `A`, `B` or `C` through `Base*` and identify the real type. |

## Build

```sh
cd ex00
make
./convert 42.0f
make fclean
```
